# Apoorva Busunur Mallikarjuna — Personal Academic Website

This repository contains the source code for my personal academic and professional portfolio website.

The website presents my background, research interests, professional experience, technical projects, scientific engagement, publications, and academic references. It is designed to showcase my interdisciplinary work at the intersection of **Aerospace Engineering, Embedded Systems, Human Factors, and Neuroergonomics**.

🌐 **Website:** https://apoorvabm6.github.io

---

## Technologies

The website is built using:

* **Jekyll** — Static site generator
* **Liquid** — Templating language
* **HTML5** — Website structure
* **Sass / SCSS** — Styling
* **JavaScript** — Interactive functionality
* **Markdown** — Content management
* **YAML** — Configuration and metadata
* **Git & GitHub** — Version control and hosting
* **GitHub Pages** — Website deployment

The website also uses **SVG icons** for social links and document links to provide scalable and lightweight graphics.

---

## Content Management and Customization

The website uses Markdown files for most textual content. The following files control the main parts of the website:

### Page Sections

```text
_content/
```
Contains Markdown content for the major sections of the website.

### Projects

```text
_projects/
```
Each project is maintained as an individual Markdown file and rendered through the Jekyll `projects` collection.


### Images and Documents

```text
assets/img/
assets/files/
```
The `assets/img` directory contains website images, while `assets/files` contains downloadable documents such as CVs and resumes.

## Running the Website Locally

### Prerequisites

You need the following installed:

* Ruby
* RubyGems
* Bundler
* Jekyll

Check the installations using:

```bash
ruby -v
gem -v
bundle -v
jekyll -v
```

---

### Install Dependencies

Clone the repository:

```bash
git clone https://github.com/ApoorvaBM6/ApoorvaBM6.github.io.git
```

Navigate into the project:

```bash
cd ApoorvaBM6.github.io
```

Install the required Ruby dependencies:

```bash
bundle install
```

---

### Run the Development Server

Start the local Jekyll server:

```bash
bundle exec jekyll serve
```

The website will be available at:

```text
http://localhost:4000
```

If a custom port is configured:

```text
http://localhost:3000
```

The local server automatically rebuilds the website when source files are modified.

---

## Building the Website

To generate the static website manually:

```bash
bundle exec jekyll build
```

The generated website will be placed in:

```text
_site/
```

The `_site` directory contains the final static HTML, CSS, JavaScript, images, and other files generated by Jekyll.

---

## Publishing to GitHub Pages

The website is hosted using GitHub Pages through the repository:

```text
ApoorvaBM6.github.io
```

The standard workflow for updating the website is:

```bash
git add .
git commit -m "Update website"
git push origin main
```

GitHub Pages then builds and publishes the updated website.

The live website is available at:

```text
https://apoorvabm6.github.io
```
